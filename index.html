Manejador de tareas

Sitio para la gestion de tareas personales de un usuario. Las tareas respetan la siguiente tipologia:
- Hogar.
- Trabajo.
- Hábito.
El proyecto soporta una definicón de prioridades de las tareas de usuario creadas. El ámbito de prioridades de tareas es la siguiente:
- Alta.
- Media.
- Baja.
La solución tambien permite :
- Crear tareas.
- Editar tareas.
- Eliminar tarea.
- Marcar tarea como realizada.
Ademas tenemos dos botones para desplazar hacia arriba o hacia abajo una tarea en la tabla. El proyecto se conecta a una base de datos postgresql.

Sistema de exámenes

Sistema de administración de exámenes con rol de administrador y rol de usuario. Permite registarse como usuario e iniciar sessión.
El sistema permite en el rol de administrador:
- Ver perfil del administrador.
- Agregar categorias.
- Listar categorias disponibles.
- Agregar exámenes.
- Actualizar exámenes.
- Eliminar exámenes.
- Modificar exámenes.
- Activar o desactivar exámanes.
- Agregar preguntas a los exámenes.
- Darle una categoría a un exámen.
- Actualizar pregunta.
- Eliminar pregunta.

El rol de usuario permite:
- Ver exámenes activos.
- Ver exámenes por categoría.
- realizar exámen.
- Enviar cuestionario.
